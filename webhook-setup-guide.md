# প্রোডাকশন সার্ভারে Razorpay Webhook সেটআপ করার নিয়ম

আপনার কোডবেস (Backend) সম্পূর্ণভাবে প্রোডাকশন সার্ভারের জন্য রেডি করাই আছে। যখন আপনি আপনার প্রজেক্টটি কোনো লাইভ সার্ভারে (যেমন Render, Vercel, VPS বা cPanel) আপলোড করবেন, তখন শুধু নিচের ধাপগুলো অনুসরণ করবেন:

### ১. .env ফাইল আপডেট করুন
আপনার লাইভ সার্ভারের `.env` ভ্যারিয়েবলের সেকশনে গিয়ে এই লাইনটি যোগ করুন:
```env
RAZORPAY_WEBHOOK_SECRET=আপনার_দেওয়া_একটি_কঠিন_পাসওয়ার্ড
```
**(যেমন: A6ZYvMviry@xL9t)**

### ২. Razorpay ড্যাশবোর্ডে লিংক সেটআপ
1. আপনার Razorpay অ্যাকাউন্টে লগইন করুন।
2. বামদিকের মেনু থেকে **Settings -> Webhooks**-এ যান।
3. **"Add New Webhook"**-এ ক্লিক করুন।
4. **Webhook URL:** এখানে আপনার লাইভ ওয়েবসাইটের ব্যাকএন্ডের ডোমেইনের সাথে ওয়েবহুক লিংকটি দেবেন।
   * উদাহরণস্বরূপ: যদি আপনার লাইভ API লিংক হয় `https://api.my-event-site.com`, তাহলে বক্সে দেবেন:
     `https://api.my-event-site.com/api/payments/razorpay-webhook`
5. **Secret:** এখানে আপনার সেই `.env` ফাইলে রাখা কঠিন পাসওয়ার্ডটি হুবহু কপি করে বসাবেন।
6. **Active Events:** নিচের দুইটি অপশনে টিক (✓) দেবেন:
   * `order.paid` (বা `payment.captured`)
   * `payment.failed`
7. সেভ করুন।

ব্যাস! আপনার প্রোডাকশনে অটোমেটিক টিকিট বুকিং ও কোয়ান্টিটি আপডেট সিস্টেম কাজ করা শুরু করবে।
